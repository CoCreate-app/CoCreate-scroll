CoCreate-scroll

A simple HTML5, CSS and pure javascript component. Easy configuration using data-attributes and highly styleable.

Reference

This is xxxxx reference

                        
<div data-scroll_up_class="up" data-scroll_down_class="down" data-scroll_ing_class="scrolling" data-scroll_range_class="range" data-scroll_range_begin="1000" data-scroll_range_end="1500" data-scroll_intersect="intersect">

Scroller div must have "cocreate-scroll" class.

Attributes

  • data-scroll_up_class string optional

    Scroll-Up-Class

  • data-scroll_down_class string optional

    Scroll-Down-Class

  • data-scroll_ing_class string optional

    Scroll-Range-Class

  • data-scroll_range_class string optional

    Scroll-Range-Class

  • data-scroll_range_begin number optional

    Scroll-Range-Begin

  • data-scroll_range_end number optional

    Scroll-Range-End

  • data-scroll_intersect string optional

    Scroll-Intersect
